About the template

Pintu MultiPurpose template for every type of website, including 15 different page layouts to build your own website.

Pintu HTML5 Template can be used for multi-purpose websites. Wether you want to create a ecommerce shop, a business website, a blog or even a personal portfolio, it is always the right solution. Promentium is a template for portfolios and ecommerce websites. It is a highly suitable template for all types of websites. It has flat and purpose oriented design, responsive layout and special features like appointment forms, as well as blog layouts and landing pages. Use easy template options and shortcodes to customize your website. This template is powered by Twitter Bootstrap v4 (the most popular front-end framework). It means that it will be easier for you to do the changes and to increase its possibilities, thanks to many plugins it possesses.

Installation

Pintu - MultiPurpose Platform HTML5 template does not need a real installation. Just upload all files included in folder ./pintu-html to your server.

Upload Template files

You have two ways to upload Pintu Template files:

  1. Upload Pintu files in folder ./pintu to your server via FTP
    • Unpack the Template .zip file (pintu-html.zip) you have downloaded.
    • Upload the /pintu folder on your server.

Files

Pintu Template includes many files, within which you'll find the following directories and files, logically grouping common resources and providing both finding and changes.

You will find 3 folders in your .zip file.

Including a documentation, the HTML template and a PSD files folder containing additional .PSD files, you can be sure that everything is completed.

Documentation

Including four different folders with coded stuff and one index.htm for you to open. Just click on the index.html and open it with any browser. Now you can see the documentation in your browser.

JavaScript

All JS files are in assets/ directory.

  • assets/libs/jquery/jquery.js - jQuery [version 3.1.0]

    It is a JavaScript library designed to simplify the client-side scripting.

  • assets/libs/jquery-ui/jquery-ui.js - jQuery UI [version 3.1.0]

    A curated set of user interface interactions, effects, widgets, and themes built on top of the jQuery JavaScript Library

  • assets/libs/slick/slick.min.js - Slick Carousel

    Slick Carousel Plugin

  • assets/libs/isotope/isotope.pkgd.min.js - Isotope

    Isotope Plugin

  • assets/libs/visible/visible.js - VisibleJS

    Identify visible elements tool

  • assets/js/main.js - Main template JS file

CSS

Below, we present the list of the most important Pintu CSS files.

  • assets/lib/slick/slick.css - dedicated Slick styles

    Stylesheet for the Slick Carousel

  • assets/libs/bootstrap/css/bootstrap.css - Bootstrap v4.0.0-beta.2

    Bootstrap stylesheet.

  • assets/css/main.css - dedicated Pintu styles

    Main template stylesheet.

PSD

You will find all .PSD files in this folder. These files can be opened with Adobe Photoshop CS1 or higher. These files does not include any images, as these are just designed layouts for the template.
